# Contributing to Deoria Shopping

## How to Contribute

### 1. Report Bugs
- Issue create karein
- Bug ka clear description dein
- Steps to reproduce batayein

### 2. Suggest Features  
- Naye ideas suggest karein
- Improvements batayein

### 3. Add New Shops
- Nayi dukaan ki information dein:
  - Dukaan ka naam
  - Category
  - Address
  - Phone number
  - Photos (agar available)
